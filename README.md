# Machine-Health-And-Maintaince
Machine learning project for predicting defect counts in industrial machines using sensor and production data. Implements preprocessing with pipelines, compares Linear Regression and Random Forest models, and evaluates performance using MAE and RMSE, along with feature importance analysis for insights.

## Description
This project predicts defect count using machine sensor data.

## Models Used
- Linear Regression
- Random Forest

## Features
- Temperature
- Vibration
- Pressure
- Energy Consumption
- Production Units
- Plant
- Machine ID

## Metrics
- MAE
- RMSE

## How to Run
```bash
pip install -r requirements.txt
python defect_prediction.py
